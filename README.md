## Kurulum

### Backend (Flask API)

1. Gerekli paketleri yükleyin:
```bash
pip install flask flask-cors
```

2. Backend'i başlatın:
```bash
cd backend
python app.py
```

Varsayılan olarak `http://localhost:5000` adresinde çalışacaktır.

### Frontend (Next.js)

1. Gerekli paketleri yükleyin:
```bash
cd frontend
npm install
# veya
pnpm install
```

2. Frontend'i başlatın:
```bash
npm run dev
```

Frontend varsayılan olarak `http://localhost:3000` adresinde çalışacaktır.
